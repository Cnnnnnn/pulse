# v2.37.0 — I8 基金盈亏阈值提醒

> 发版日期: 2026-06-24
> 主分支: main

## 新增

- **I8 盈亏阈值提醒**: 基金管理 Header 🔔 按钮配置全局阈值
  - 盈利提醒: 持仓收益率 ≥ 设定值 (默认 +10%)
  - 亏损提醒: 持仓收益率 ≤ 设定值 (默认 -5%)
  - 净值定时/手动刷新后检查，越过阈值发系统通知
  - 中性区间自动重置；同方向需再越过 2pp 才重复提醒
  - 尊重通知静默时段 (`notifications.quiet_hours_*`)
  - 偏好持久化到 `state.json.funds.alertPrefs` (默认关闭)

## 范围说明 (v1)

- 全局阈值，不做单只基金独立配置
- 不做定投计划 / 推送频率自定义
- 通知走 Electron 系统通知，无应用内角标

- [ ] 用户手测: 开启提醒 → 设低阈值 → 刷新净值 → 收到通知
