# v2.38.0 — I2 v2 Watchlist 扩展

> 发版日期: 2026-06-24
> 主分支: main

## 新增

- **I2 v2 关注列表**: `watchlist` 项支持 `type` + `ref`
  - **app** (v1 兼容): 升级通知，旧 `appName` 自动迁移
  - **fund**: 基金行 ☆/★ pin，净值相对上次提醒变动 ≥2% 时通知
  - **keyword**: 抽屉内添加关键词，微博热搜 / IT之家标题 substring 匹配
- 基金净值刷新、热搜刷新、IT之家新闻刷新后自动跑 keyword/fund checker
- 共享 `makeWatchlistSendNotification` + 静默时段

## 范围说明 (v2)

- 贵金属 pin 留后续版本 (metal type 未接入 UI)
- 关键词按标题 substring，不做全文搜索
- 基金首次 pin 只记基准净值，不立即通知

## 验证

- [x] vitest: watchlist v2 pure/side-effect + IPC + WatchlistDrawer
- [ ] 用户手测: 基金 pin → 刷新净值 → 大幅波动时通知；抽屉加关键词 → 刷热搜
