# Design Context — 基金管理系统重设计

> Established via `impeccable` skill. Persisted for all future sessions of this redesign.

## Users
基金从业者与投资者（个人 / 机构），在桌面与移动端监控资产、浏览与筛选基金、查看详情、执行申购 / 赎回。
语境：日常盯盘、定期复盘、临场决策。
情绪目标：**信任、沉稳、掌控感（confidence · calm · control）**，而非兴奋或花哨。

## Brand Personality
专业 · 克制 · 可信（Professional · Restrained · Trustworthy）。金融级清晰度，数据优先。

## Aesthetic Direction
- 简洁专业金融风；**蓝绿色系（teal / 蓝绿）为主强调色**，配合深墨蓝（ink）作主操作色，刻意避开“青 + 纯黑发光”套路。
- **浅色为默认主题**（金融舒适），深色为次级，两者均支持切换并持久化到 localStorage。
- 编辑式金融排版：小号大写 tracked 标签、tabular 数字、发丝级分隔线、留白节奏。
- 反参考：纯黑底 + 霓虹青发光、紫蓝渐变、英雄指标模板堆砌、圆角 + 通用重阴影。

## Design Principles
1. **数据优先**：数字用 `tabular-nums`、清晰层级；图表交互流畅且有教育意义（hover 提示、区间切换）。
2. **克制的层次**：发丝边框 + 极柔阴影，靠留白与排版建立层级，不靠重阴影。
3. **双主题一致**：同一套 token 驱动浅 / 深，对比度满足 WCAG AA（正文 ≥ 4.5:1，大字 ≥ 3:1）。
4. **流畅但克制的交互**：状态化动效用 `ease-out-quart` / `expo`，尊重 `prefers-reduced-motion`；加载 / 成功 / 错误均有明确反馈。
5. **响应式不降级**：移动端侧栏折叠为抽屉、表格转为卡片态，关键功能不隐藏。

## Deliverable
独立、自包含、零外部依赖的 HTML 原型（`fund-management-redesign/`）：hash 路由切换 4 个模块 + 外壳，内联 SVG 图表，可直接浏览器打开预览。

## Tech Notes
- 原生 HTML + CSS + 非模块 JS（`<script src>`，兼容 `file://` 直接打开）。
- 排版：中文 PingFang SC / Microsoft YaHei + 拉丁 system-ui；数字 tabular-nums。
- 颜色：oklch 令牌，浅 / 深双套。
