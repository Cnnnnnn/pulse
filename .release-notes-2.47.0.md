# v2.47.0 — 📈 选股分析 (阶段一): A 股条件选股筛选器

> 全新方向 (v2 路线图未覆盖的新功能): 通过条件筛选全市场 A 股, 找出符合策略的票, 存自选 + 后台行情刷新.
> 数据源走东财接口 (纯 JS, 无新依赖), 完全对照现有基金模块的架构.

## 新增

- **📈 选股 tab** — 条件选股筛选器
  - 4 个内置策略 (chip 横条一键切换): 低估值高ROE / 蓝筹白马 / 高股息 / 成长动量
  - 选策略自动填条件, 仍可在条件区微调 (微调即切"自定义")
  - 4 类筛选维度: 估值财务 (PE/PB/ROE/股息率) + 行情动量 (涨跌/换手) + 行业 + 市值区间 (大/中/小盘)
  - 结果表格: 列头可点排序, 涨跌红涨绿跌 (A 股惯例), 每行 ⭐ 存自选
  - 条件区精简 (默认露 PE/ROE/市值, ⚙ 高级折叠 PB/股息/换手/动量)
- **⭐ 自选股 tab** — 自选股列表 + 行情
  - 搜索代码或名称添加 (+ 添加 Modal, 模糊搜, 自动填名称)
  - 盘中后台定时刷新行情 (独立 StockQuoteScheduler, 9:30-15:00 工作日每 5 分钟)
  - 实时价 + 涨跌% + PE + ROE, 推送到渲染端

## 设计要点

- **数据源**: 东财 `push2.eastmoney.com/api/qt/clist/get` 单接口拉全市场 (约 5000 只) 全字段
- **null 跳过语义**: 某只票某字段缺失 (如 pe=null) → 跳过该条件而非排除, 避免误删
- **节流**: 筛选结果 60s 内存缓存 (避免连点重复打接口); 自选股后台 5 分钟固定间隔
- **独立 scheduler**: StockQuoteScheduler 不复用 FundScheduler, 避免两套数据耦合
