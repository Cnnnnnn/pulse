# v2.40.0 — I2 贵金属 pin + Q4 启动优化 + A4 用量异常

> 发版日期: 2026-06-24
> 主分支: main

## 新增

### I2 v3 贵金属 Watchlist pin

- 贵金属卡片 ☆/★ pin，价格相对上次提醒变动 ≥2% 时系统通知
- `watchlist` 支持 `type: metal`，`ref` 为 `XAU` / `XAG` / `AU9999` / `AG9999`
- 金属 scheduler 刷新报价后自动跑 checker（与基金 watchlist 对称）

### A4 AI 用量异常检测

- 7 日 sparkline 上检测今日用量尖峰（≥ 前 6 天中位 × 1.5 且 ≥ 55%）
- AI 用量页异常 banner + 今日柱高亮
- 30 分钟 refresh 后自动通知（含疑似高消息量任务 top-1）
- 去重状态 `state.json.ai_usage_alert_prefs`

## 优化 (Q4 v3)

- **baseline**: main 冷 require 中位 ~20ms（`npm run baseline:q4`）
- **延后 AI 用量 warmup** 到 `setImmediate`，不阻塞 bootstrap 同步段
- **延后 tray refresh 首次 fetch** 到 `setImmediate`（`deferInitial: true`）
- renderer bundle ~941KB；若 `readyMs` 仍 >800ms 需本地 DevTools profile（`scripts/q4v2-profile.sh`）
