# v2.36.0 — A2 升级建议 + A7 v2 Prompt 补全

> 发版日期: 2026-06-24
> 主分支: main

## 新增

- **A2「该不该升」AI 建议**: 有更新的 app 行显示「💡 该不该升?」按钮, 按需调用 LLM 分析
  - 输入: release notes + 安装/最新版本 + 使用频次 (last-opened hot/warm/cold)
  - 输出 JSON: `recommendation` (upgrade/wait/skip) + `summary` + `reasons`
  - 按 `appName::latestVersion` 缓存到 `state.json.upgrade_advice_cache` (30 天 GC)
- **A7 v2 Prompt 补全**:
  - 新增 `upgrade_advice` + `category_classify` 两个可配置 prompt
  - Few-shot 示例字段 (可选 textarea)
  - 每项「恢复默认」按钮 (`ai-prompts:reset` IPC)
  - `classifyByLLM` 改用 `resolvePrompt('category_classify')`

## 优化

- `ithome_summary` 支持 few-shot 注入 system

## 范围说明 (v1)

- 升级建议需配置 AI API Key (共用 ai_sessions_config)
- 不做批量分析 / 自动后台预取 (点击才调 LLM)
- category prompt 的 `{{CATEGORY_IDS}}` 占位符在运行时替换

## UX (v2.36.1 同版)

- AI 任务抽屉 Header 新增 ⚙️ 设置按钮 (高亮=设置页, 再点返回列表)
- 设置页拆 **连接设置** / **Prompt 模板** 两个 Tab, 底部 footer 也可直达
- 侧栏 **🤖 AI 配置** 弹窗同步接入相同 Tab (与任务抽屉共用 `AISettingsScene`)
