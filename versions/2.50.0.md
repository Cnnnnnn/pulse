# v2.50.0 — 🗑 移除股票「自选股」tab 与周边能力

> v2.47.0 引入的股票自选股（侧栏 `⭐ 自选股` tab）实际使用频率低，与「选股筛选器」重复且容易让用户混淆。整段移除：UI、后台 5min 行情推送、IPC、持久化字段、单元测试。

## 移除

- **侧栏 `⭐ 自选股` nav tab** — `SideNav` / `LazyNavPanel` 不再注册 `stock-watchlist`
- **自选股 tab 视图** — `src/renderer/stocks/WatchlistPanel.jsx` + `AddStockModal.jsx` 整文件删除
- **选股结果表的 ⭐ 列** — `ResultTable` 移除 `isInWatchlist / addWatchlist / removeWatchlist / IconStar`，每行尾部不再有存自选按钮
- **`StockLayout` mount 时副作用** — `loadWatchlist / subscribeWatchlistQuotes / refreshWatchlistQuotes` 全部移除
- **`StockQuoteScheduler`** — `src/main/stocks-scheduler.js` 整文件删除（独立实例，独立 IPC channel）；bootstrap + `index.js` 不再启动
- **5 个 IPC 通道** — `stocks:watchlist:list / :add / :remove / :quotes` + `stocks:watchlist:quotes` 推送事件，全部移除
- **`state.json.stockWatchlist` 字段** — `PRESERVE_FIELDS` 不再保留；下次写盘自然消失
- **`src/main/stock-store.js`** — 自选股 + screener prefs 持久化模块整文件删除
- **5 个 stock watchlist API 桥接** — `preload.js` + `src/renderer/api.js` 移除 `stocksWatchlistList/Add/Remove/Quotes` + `onStocksWatchlistQuotes`

## 改动

- `src/renderer/worldcup/navStore.js` — `NAV_KEYS` / `NAV_KEYS_LIST` 移除 `stock-watchlist` (10 → 9 项)
- `src/main/index.js` — 移除 `StockQuoteScheduler` import + `startStockScheduler` 调用
- `src/main/bootstrap/schedulers.js` — 移除 `startStockScheduler` 函数 + module 导出
- `src/main/ipc/register-stocks.js` — 移除 4 个 watchlist handler + `stockStore` import + `fetchStocksByCodes` import
- `src/main/state-store.js` — `PRESERVE_FIELDS` 移除 `stockWatchlist`
- `src/renderer/stocks/{StockLayout,stockStore,ResultTable}.jsx` — 移除 watchlist 订阅 / signal / helpers
- `src/renderer/components/{SideNav,LazyNavPanel}.jsx` — 移除 `stock-watchlist` 项 + loader
- `tests/main/register-stocks.test.js` — 移除 4 个 watchlist case + 相关 mock
- `tests/renderer/sidenav-prefs.test.{js,jsx}` — 期望数组去除 `stock-watchlist`，长度从 10 → 9
- `tests/main/{stock-store,stocks-scheduler}.test.js` — 整文件删除

## 保留

- **应用库 watchlist**（`src/renderer/watchlist/`, `state.json.watchlist`）— 跟「关注 app」有关，跟股票无关，**不动**
- **stockScreener 字段**（上次条件 / 策略 / 排序）— 是选股筛选器偏好，跟自选股无关，**不动**
- **个股分析** tab（`stock-detail`）— 跟自选股无关，**不动**
- **AI 推荐抽屉 / AI 个股分析** — 行情拉取链路不依赖自选股，**不动**

## 不在本次范围

- `state.json.stockWatchlist` 老字段不再 preserve，但旧文件残留字段不会自动删除（无害，下次 `patchState` 时被覆盖）。需要彻底清理可走 P61 配置导出/导入。
- `docs/superpowers/specs/2026-06-26-stock-screener-design.md` §3.3 自选股章节保留作为历史设计记录。
