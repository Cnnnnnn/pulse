React

React is a JavaScript library for building user interfaces. It lets you compose complex UIs from small, isolated pieces of code called "components".

Installation

npm install react react-dom

Quick Start

import { createRoot } from 'react-dom/client';
createRoot(document.getElementById('root')).render(<App />);

Key Features

  • Declarative: design simple views for each state.
  • Component-Based: build encapsulated components that manage their own state.
  • Learn Once, Write Anywhere: develop for web, native, and VR.

See the official documentation for the full guide.

一句话摘要
业界主流的声明式 UI 构建库，用组件模型降低大型前端应用的维护成本，是 React 生态（Next.js、Expo）的事实标准底座。
核心功能
  • 声明式渲染：状态驱动视图，自动 diff 更新 DOM
  • 组件化：可复用、可组合的 UI 单元
  • Hooks：用函数组件管理状态与生命周期
  • 跨平台：Web / React Native / VR 同一套心智
适用场景
  • 中大型单页应用（SPA）与后台管理系统
  • 需要长期维护、团队协作的前端项目
  • 跨端产品（Web + 移动）希望复用逻辑
使用方法
通过 npm install react react-dom 引入，配合构建工具（Vite/Next.js）即可启动；优先用函数组件 + Hooks 组织代码，复杂状态可引入 Zustand/Redux。
