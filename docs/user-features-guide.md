## 一、应用概览

Pulse 是一款 **macOS / Windows 菜单栏常驻小工具**,核心能力是监控你本地安装的一批应用的版本,有新版即提醒并一键升级。在此之上扩展了资讯、行情、AI 等多个独立面板。

整体布局:

```
┌──────────┬───────────────────────────────────┐
│          │  顶部工具栏 (随 Tab 变化)            │
│  左侧    ├───────────────────────────────────┤
│  导航    │                                   │
│  SideNav │   主内容区 (随 activeNav 切换)      │
│ (可折叠) │                                   │
│          │                                   │
└──────────┴───────────────────────────────────┘
```

- **左侧导航 (SideNav)**:7 个 Tab,支持折叠 (180px ↔ 40px)、拖拽重排、隐藏/恢复、未读角标。
- **顶部工具栏**:每个 Tab 有完全独立的 Header 和操作按钮。
- **全局抽屉/弹窗**:Reminders、Recent、Watchlist、Diagnostics、AI Tasks、Search 等,跨 Tab 可用。

## 二、左侧导航 Tab 功能

### 1. 版本检查(核心 Tab)

> 入口:🔄 · 应用主功能 · 检测本地 app 是否有新版本

**顶部工具栏 (Header) 操作:**

| 按钮 | 功能 |
|------|------|
| **检查更新** | 触发全量检测,并发拉取每个 app 的最新版本(WorkerPool 4 线程) |
| **JSON / CSV** | 导出当前检测结果到桌面文件 |
| **一键升级 (BulkUpgradeButton)** | 打开批量升级弹窗 |
| **🤖 AI 任务** | 打开 AI 任务总结抽屉 |
| **⏰ 提醒** | 打开提醒弹窗 |
| **🕒 最近活动** | 打开跨 Tab 时间线 |
| **🔧 诊断** | 打开错误诊断抽屉 |
| **★ 关注列表** | 打开关注列表抽屉 |
| **版本日志** | 打开 Release Notes |

**筛选区 (FilterBar):**
- 搜索框:实时按 app 名称/bundle 过滤(Esc 清空)
- 4 个状态 tab:全部 / 有更新 / 已是最新 / 出错(带计数)

**分类 Tab (CategoryTabs):**
- 按 8 个分类过滤 app:AI 工具 / 开发者 / 浏览器 / 沟通 / 媒体 / 笔记 / 系统 / 其他
- 数字键 `0-9` 快速切换

**单行 (AppRow) 操作:**

| 操作 | 说明 |
|------|------|
| 点击行 | 打开该 app 的官方下载页 |
| **升级** 按钮 | 单 app 一键升级(macOS `brew upgrade` / Windows `winget upgrade`) |
| **⏰ 延后** | 打开 snooze 菜单:今晚 / 明早 / 周末 / 跳过此版本 |
| **⏪ 回滚历史** | 打开版本历史抽屉,查看旧版备份并一键回滚 |
| **★ 关注** | 加入/移出关注列表 |
| **右键** | 打开静音菜单(按使用频率智能推荐时长:1/7/30/90 天/永远) |
| ⓘ 信息 | 展开/收起 changelog 面板 |

**批量升级弹窗 (BulkUpgradeModal):**
- 按数据源分组(brew / sparkle / App Store / winget 等)
- 复选框勾选要升级的 app,默认全选可升级项
- 升级前自动备份最近 2 个版本(cap 2)
- 支持:启动 / 取消(运行中) / 单行重试 / 一键重试所有失败项 / 查看日志

**版本历史抽屉 (VersionHistoryDrawer):**
- 列出该 app 的历史备份版本 + 占盘大小
- "回滚到这版":关闭应用并用备份替换当前安装
- "删除备份":清理备份文件

### 2. IT 新闻

> 入口:📰 · IT之家资讯聚合 + AI 摘要

**功能:**
- **本月新闻 / 收藏** 两个子 Tab
- **刷新** 按钮:拉取当前日期 IT之家资讯
- **搜索框**:按标题、分类过滤(`Cmd+F` 聚焦)
- **左侧日期栏**:按日期浏览历史新闻
- **收藏**:⭐ 收藏文章,永久保留
- **AI 摘要**:对新闻生成 AI 总结(需配置 AI Provider)
- **未读角标**:导航栏显示未读数

> 入口:🔥 · 微博实时热搜榜(手动刷新)

### 5. 基金管理

> 入口:💰 · 基金持仓 + 实时盈亏监控

**总览卡片 (FundHeader):**
- 总市值 / 总盈亏 / 今日预估 / 总成本 / 本月累计(对比上月)

**工具栏:**
- **净值数据源切换**:天天基金 / 其他源
- **搜索框**:按代码/名称过滤
- **+ 添加持仓**:打开添加/编辑基金弹窗
- **🔔 阈值提醒**:配置盈亏阈值,到点系统通知
- **🔄 刷新**:立即拉取最新净值

**2 个主 Tab:**
- **持仓**:基金列表 + 分类过滤(全部/股票/债券/货币/QDII/其他)
- **盈亏记录**:历史盈亏曲线

**添加/编辑基金弹窗:**
- 输入基金代码、名称、持仓份额、成本净值
- 自动拉取实时净值计算盈亏

### 7. AI coding plan 用量

> 入口:📊 · Minimax / GLM 编程套餐配额监控

**功能:**
- **双 Provider Tab**:Minimax / GLM 各自独立
- **配额卡片**:5 小时滚动窗口 / 周窗口 / 视频/MCP 配额
- **进度条 + 倒计时**:每个窗口的已用百分比 + 距重置时间

## 三、跨 Tab 全局功能

### 提醒 (Reminders)

> 入口:Header ⏰ 按钮 / 快捷键 `⌘⇧R`

纯本地轻量提醒工具,不联动其他模块。

- **新建提醒**:标题 + 触发时间 + 4 种重复(一次/每天/工作日/每周某天)
- **触发**:到点系统通知 + Header 红色 badge,状态切 `fired`(待 ✓ 打卡)
- **完成**:一次性提醒自动删;重复规则算下次时间,自动切回 pending
- **管理**:编辑 / 删除 / 忽略 / 打卡完成

### 最近活动 (Recent)

> 入口:Header 🕒 按钮

跨 5 个 Tab 的统一时间线,倒序展示"最近我做了什么":
- 升级 / 检查 / 提醒 / 比赛 / 基金 / 新闻 / 设置 等操作记录
- 按类别过滤(全部/升级/提醒/比赛/基金/新闻/设置)
- 点击条目跳转到对应 Tab
- 容量受 `config.json.recentActivity.maxEntries` 控制(默认 200)

### 关注列表 (Watchlist)

> 入口:Header ★ 按钮 / 各卡片 ⭐

统一的关注中心,聚合 4 类关注项:
- **App**:版本检查列表里 ⭐
- **基金**:基金卡片 ⭐
- **贵金属**:贵金属卡片 ⭐
- **关键词**:手动添加(热搜/IT新闻匹配)

每项记录:基准值 + 最近通知值 + 添加时间。

### 错误诊断 (Diagnostics)

> 入口:Header 🔧 按钮

开发/排错向的抽屉:
- 错误日志列表(按级别统计)
- 启动耗时指标
- 高频失败项 Top N
- 采样数据
- 操作:复制全部 / 打开日志文件夹 / 清理旧日志 / 导出

### AI 任务总结 (AI Tasks)

> 入口:Header 🤖 按钮

扫描本地 AI 编程工具(Cursor / Codex / MiniMax Code)的会话,生成每日总结:
- **按应用分组**展示任务卡(标题/项目/时间段/消息数/状态)
- **日期切换**:今天/昨天/前 7 天快捷 chips
- **生成总结**:勾选任务 → 调 LLM 生成"用户诉求 / 处理结果"
- **状态**:未总结 / 生成中 / 已总结 / 内容已更新
- **查看原始**:跳转到源 app 打开会话
- 需先在 AI 设置里配置 Provider 和 API Key

### 全局搜索 (Search)

> 入口:快捷键 `⌘K` / `⌘⇧K`

跨模块全文搜索:
- 输入框 + 左侧来源栏 + 右侧结果列表
- 来源:新闻 / AI 任务 / 提醒 等
- 键盘导航:↑↓ 选择 / Enter 跳转 / Esc 关闭

### 批量升级 (Bulk Upgrade)

> 入口:Header "一键升级" 按钮

见 [版本检查 Tab](#1-版本检查核心-tab) 的批量升级弹窗说明。

## 四、键盘快捷键汇总

| 快捷键 | 功能 |
|--------|------|
| `⌘F` | 聚焦当前 Tab 的搜索框 |
| `⌘K` / `⌘⇧K` | 打开/关闭全局搜索 |
| `⌘⇧R` | 打开提醒新建表单 |
| `⌘⇧F` | 跳转到基金管理 Tab |
| `⌘⇧M` | 跳转到贵金属 Tab |
| `0-9` | 版本检查 Tab:切换分类(0=全部) |
| `Esc` | 清空搜索 / 关闭弹窗 / 关闭菜单 |
| `↩` (搜索结果) | 跳转到选中项 |

> 注:`⌘` 在 Windows 上对应 `Ctrl`。

## 五、设置与配置

### AI 配置 (SideNav 底部 🤖 按钮)

- **连接设置**:Provider / Model / API Key
  - 支持:OpenAI / Anthropic / DeepSeek / MiniMax / GLM(智谱)
  - API Key 走 Electron `safeStorage`(macOS Keychain / Windows DPAPI)加密存储
- **Prompt 模板**:自定义 AI 总结的 prompt
- **Token 预算**:配置每日 token 上限,超额拦截(当前开发分支功能)

### 托盘菜单配置 (TrayMenuConfigModal)

> 从菜单栏右键/设置入口打开

- 6 个 checkbox 控制:更新 / AI 用量 / 世界杯 / 贵金属 等栏目在系统托盘菜单中的显隐
- 主面板 SideNav 与托盘菜单联动(关掉的栏目两边都隐藏)

### 侧边栏自定义

- **折叠/展开**:☰ 按钮切换(180 ↔ 40px)
- **重排**:拖拽 Tab 项调整顺序
- **隐藏**:右键隐藏不常用 Tab
- **恢复**:"已隐藏 (N)" 入口打开抽屉恢复

### app 配置

编辑 `config.json`(应用关闭后改,下次启动自动重读):
- 每个 app 配置 `name` / `bundle` / `download_url` / `version_sources` / `detectors`
- 详见 README 的"加/改 app"章节

## 六、数据持久化与文件位置

| 数据 | 位置 |
|------|------|
| 状态缓存 (last-known 结果) | `~/Library/Application Support/pulse/state.json` (macOS)<br>`%APPDATA%/pulse/state.json` (Windows) |
| AI API Key | `~/Library/Application Support/pulse/ai-keys/<provider>.bin` (mode 0600) |
| 提醒 / 最近活动 | 写入 `state.json` |
| 关注列表 / 静音 / snooze | 写入 `state.json` |
| 版本备份 | 升级时自动保存最近 2 版(可回滚) |
| app 配置 | 项目根 `config.json` |

> 删除 `state.json` 等于清空所有缓存和本地数据(不影响 app 配置)。
