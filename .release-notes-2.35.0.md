# v2.35.0 — C7 检测结果导出 (JSON / CSV)

> 发版日期: 2026-06-24
> 主分支: main

## 新增
- **检测结果导出**: Header 区新增 JSON / CSV 两个按钮,一键把当前 last-known 检测结果写到桌面
  - JSON: `pulse-detect-results-{ts}.json` — 含 `generatedAt` / `pulseVersion` / `count` / `apps[]`
  - CSV: `pulse-detect-results-{ts}.csv` — 10 列核心字段 (name / 版本 / status / source / ts 等)
  - 数据源: `state.json` apps (跟 UI 展示一致, 不含 trace / changelog 等大字段)
- **主进程模块** `detect-results-export.js`: 纯函数 `pickExportFields` + `toCsv` + `exportDetectResults`
- **IPC** `detect-results:export` — preload / api.js 桥接

## 优化
- 零 (独立功能, 不碰检测链路)

## 范围说明 (v1)
- 只导出 state 里已有的 last-known 结果, 不做历史快照 / 定时自动导出
- 写出目录固定 ~/Desktop (跟 diagnostics-export 一致)
- 不做 Excel (.xlsx) / 自定义列选择 (留 v2)
